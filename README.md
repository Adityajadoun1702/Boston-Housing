# Boston Housing Project
This is a multiple linear regression model for Boston Housing datasheet from kaggle
## Technology Used:
Simple Linear regression
checked linearity using residual plots
checked constant variance using residual plots
checked Autocorrelation using DW test
checked multivariate normality using histogram
checked multicolinearity using VIF
## Git commands used to implement Version Control
git init: creating a git repository
git config --global --list: global configuration applied to all repository of current user
git config --local --list: local configuration applied to only current repository of current user
git add . : staging the repository
git commit -m commit name : committing the repository
git status: checking the status
git log: shows details of all commits
git log -oneline: shows details of all commits in one line
git remote add origin https://github.com/username/project.git: connecting remote repository to the github repository online
git remote -v: verifying the connection
git push origin master: pushing the master branch to the github repository
